# Indian Constitution Repository

## Overview
This repository contains the full text, articles, schedules, and amendments of the Constitution of India. It aims to provide a comprehensive, accessible, and well-organized collection of India's constitutional documents for educational and reference purposes.

## Contents
- Full text of the Constitution of India
- List of Articles by Part
- Schedules of the Constitution
- Amendments history and details
- Important landmark judgments related to the Constitution
- Resources for studying Indian constitutional law

## Usage
You can browse through the different parts of the Constitution, access specific articles or amendments, and use this repository as a study or reference guide. Contributions to improve the accuracy or add supplementary materials are welcome.
